# 智慧农业仓库信息管理系统

## 项目简介

本系统是一个基于Web的智慧农业仓库信息管理平台，采用现代化的前端技术栈，实现对农产品从分类、采购到出库的全流程信息化管理。

## 系统功能模块

### 1. 货品分类模块
- **功能描述**: 实现对不同种类农产品的智能分类展示
- **支持分类**: 水果、蔬菜、水产、禽畜肉蛋、农副产品、种子幼苗等主要农产品分类
- **特色功能**: 
  - 多维度检索与管理功能
  - 快速查找和统计各类农产品信息
  - 可视化分类展示

### 2. 采购管理模块
- **功能描述**: 展示商品详细信息，包括库存情况、市场价格、产地来源等
- **核心功能**:
  - 基于供应链数据和市场价格趋势分析
  - 自动生成采购建议
  - 采购计划制定和执行跟踪功能
  - 实时价格趋势图表

### 3. 爱心助农模块
- **功能描述**: 展示滞销农产品的基本信息、产地情况、销售数据等
- **主要特色**:
  - 建立消费者直接助力渠道
  - 减少中间环节成本
  - 通过信息化手段帮助农户拓展销售渠道
  - 实时进度展示

### 4. 农业资讯模块
- **功能描述**: 动态更新并展示农业相关资讯信息
- **内容类型**:
  - 市场行情
  - 行业动态
  - 政策解读
  - 农业科普
- **功能特点**: 帮助农户及时获取行业信息，做出科学决策

- **JavaScript ES6+**:

  - 动画效果实现

## 使用方法

### 1. 本地运行
1. 下载所有文件到本地目录
2. 使用现代浏览器打开 `index.html` 文件
3. 系统将自动加载并显示主界面

### 2. 功能操作
- **导航**: 点击顶部导航栏切换不同功能模块
- **分类筛选**: 在货品分类页面使用筛选按钮查看不同类别
- **采购管理**: 查看采购建议并确认采购订单
- **爱心助农**: 选择援助项目并输入援助数量
- **资讯浏览**: 在农业资讯页面查看最新行业信息

### 3. 移动端适配
- 系统完全支持移动端访问
- 自动适配不同屏幕尺寸
- 触摸友好的交互设计

## 数据库集成说明

本系统前端展示的所有数据均来自MySQL数据库，后续将提供数据库配置信息。系统设计时已预留数据接口，便于与后端数据库进行集成。

### 数据来源规划
- **货品分类数据**: 来自商品分类表
- **采购管理数据**: 来自库存表和采购记录表
- **爱心助农数据**: 来自助农项目表和销售数据表
- **农业资讯数据**: 来自资讯内容表
